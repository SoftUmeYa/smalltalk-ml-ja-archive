青木＠京都上賀茂です。

お待たせをしました。MIDIのためのクラス群を含んだ【Jun774】を
アップロードしました。カテゴリ『Jun-MIDI』の中に収録されてい
る例題プログラムを動かしてみれば、百の説明よりも多くを得るこ
とができると思います。

http://www.sra.co.jp/people/aoki/Jun/htmls/News.html

音を出せるのは Windows Vista / XP です。特に Windows XP な人
はサウンドミキサーのシンセサイザのボリュームを上げておくこと
を忘れずに。Windows Media Player が動くとそのボリュームを0に
しますので、気をつけてくださいませ。

# Mac OS X で楽しめるのは JunMidiRecorder にある例題プログラ
# ムぐらいかしらん…。音は出ませんが、MIDIファイルは作れます。

さらに詳しいことは澤本依里さんから追ってあると思います。もっ
と奇抜なプログラミング界面を年度末までに用意しますから、お楽
しみに。

# 今、研究室では、Smalltalk のプログラムが、ネット上に落ちて
# いるMIDIファイルを探してきて、エンドレスで再生（演奏）して
# います。さっきまでウィリアムテル序曲がかかっていました。現
# 在はフィガロの結婚の序曲。
