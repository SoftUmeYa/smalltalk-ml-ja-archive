日時: 2016/11/25(金) 19:00-21:00 (2時間)

東工大で研究されているMatthias SpringerさんにSqueakの新たなモジュールシステムMatrionaについて解説してもらいます

'I will talk about "Matriona", a module system for Squeak/Smalltalk,
which is similar to Newspeak. The two main ideas of Matriona are
"class parameterization" and "class nesting".
 They allow programmers to write mixins, to organize their classes
hierarchically, and to use class hierarchy inheritance. I will show
examples for both of them and explain how Matriona can be used to
write more modular Smalltalk code.'
