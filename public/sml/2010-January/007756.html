初歩的なことですが、GNU Smalltalkがインストールできません。
 
まずは下記サイトでGNU Smalltalkを知りました。
http://sites.google.com/a/smalltalk-users.jp/home/Home/gao-zhi
そこからリンクをたどっていき、下記サイトに行き着きました
http://swikis.ddo.jp/umejava/63
ここでシロクマの力とVMwareをダウンロードしましたが、PofWB-gst3.1.isoって何？
.isoファイルの取り扱い方がわかりません。VMwareはPofWB-gst3.1.vmwareというフォルダの中に
いくつかファイルがあるのですがどれも取り扱い方がわかりません。PofWB-gst3.1.vmxというファイルがキーになると思うのですが…
しかたがないのでVMwareはhttp://www.vmware.com/jp/products/player/からVMware-player-3.0.0-203739.exeというファイルをダウンロードしてきてインストールしようと思いましたが、エラーになってインストールできませんでした。そこでhttp://old.nabble.com/GNU-Smalltalk-Windows-Installer-(Beta)-td18283997.htmlからsetup.exeをダウンロードしてインストールしましたが、gstと入力しても、[could't load system file 'Bailtins.st' : No such file or directory image bootstrap failed, use option --kernel -directory]と表示されてエラーとなります。
googleでもいろいろ調べてみましたがだめでした。
 
勉強会のときは人に借りていたｐｃに正常にインストールができました。その時はシロクマもVMwareも正常にインストールしていた記憶があります。
