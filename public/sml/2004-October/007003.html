「じゅん for Java」のバージョン482を公開します。

  バージョン454からの主な変更点は、以下の通りです。

    * Java 2 Platform Standard Edition 5.0 に対応しました。 

    * JunOpenGL3dObject の典型的なオブジェクトとして、単位球(半径
      1の球)に内接する多面体を追加しました。 
              
    * 正三角形で構成される多面体として単位球を生成するメソッドを 
      JunOpenGL3dObject の static メソッドとして実装しました。 
      
    * JunOpenGLIllusion は、錯視を表現するフラックスオブジェクト
      を生成するためのクラスです。 

    * JunApplicationModelに対応する共通の抽象クラスとして、
      JunAbstractView と JunAbstractController を導入しました。 

    * Mac OS X 10.3.5 と Java 1.4.2 Update1 に対応しました。 


  「じゅん for Java」は、以下のURLからダウンロードすることができ 
  ます。

    ftp://ftp.sra.co.jp/pub/lang/java/jun4java/

  「じゅん for Java」は、オープンソースソフトウェアとして無償で提
  供している三次元グラフィックおよびマルチメディアを用いたアプ  
  リケーション構築のためのフレームワークです。「じゅん for Java」 
  に関する詳細は、以下のURLを参照してください。

    http://www.sra.co.jp/people/nisinaka/Jun4Java/index-j.html
