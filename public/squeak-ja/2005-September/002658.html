最近いくつかの重要サイトでも使われはじめているSamllWiki1ですが、日本語が
使えなくて残念に思っていた人も多いと思います。

ご協力お願いします。


試し方
------
まだCharset版はSqueakMapにあがっていないので、次の手順でおねがいします。
ベースイメージはSqueak3.8です(3.7以前ではインストールできません)。
http://ftp.squeak.org/3.8/win/Squeak3.8-current-win-full.zip

1. Squeakを立ち上げ、SqueakMapから現行のSmallWikiをインストール。
SqueakMap Package Loarderでネットワークから最新版(0.9.53)をインストール
する。
途中で出るダイアログには次のもの以外は「Ok」を選択。
「load old dependency KomService-gk.1 ?」→「No」

2. Charestが使えるバージョンをロードする。
http://www.squeaksource.com/smallwiki1/SmallWiki-chbu-0.9.53-sam.23.mcz
をIEなどでダウンロードして、Squeakの画面へドロップし「load version」を選
択。

3. SmallWikiを起動。
Squeak画面に出てきた「Welcome to SmallWiki」にしたがって普通に起動する。

以下をDoItすればとりあえず動きます。
server := SWKom startOn: 8080.
server basePath: 'http://localhost:8080'.

4. [admin:components]でCharsetを設定。
先ずは、Loginページからadminとしてログイン(パスワードはデフォルトだと
「smallwiki」)。
次に、ページ上部のメニューadmin→componentsページでCharset設定しSaveする。
(UTF-8, Shift-JIS, EUC-JPなど)

こんな感じ。
http://yengawa.homedns.org:8888/YengawaWiki/uploads/15/SW1-Charset-Components.png

5. 日本語で書き込み。
Editで日本語を書いてみる。

タイトル、内容、検索に日本語を使えるはずです。
