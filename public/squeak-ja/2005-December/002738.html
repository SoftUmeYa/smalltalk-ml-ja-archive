こんにちは、山宮です。[Squeak-ja: 2731] Squeakland 2005 J リリース計画
http://www.smalltalk.jp/pipermail/squeak-ja/2005-November/002731.html 
にてお伝えしたとおり、最新の前リリース版インストーラを公開します。全国の
スクイークファンの皆様の積極的なデバッグ参加をお待ちしております。

* ダウンロード

Windows 用
http://squeakland.jp/developer/download/installer/SqueakPluginJ-2005-11-30.exe
Mac 用
http://squeakland.jp/developer/download/installer/SqueakPluginJ-2005-11-30.dmg

- このスクイークはリリース直前のバグ修正を目的とした物です。
- 正式リリースは12月19日までお待ちください。
- バグ報告、ご要望はこのメーリングリストやバグ報告サイト 
  http://www.kukakuka.jp/squeak/bugs/ にお願いします。
- インストール後、ご利用の前に必ずアップデートをして下さい。

* アップデートの仕方。

- エスケープキーでワールドメニューを出す
- [ヘルプ] の [サーバからアップデートを取得] を選択。
- ワールドメニューの [保存] でその状態を保存する

* イメージについて

単体のイメージは 
http://squeakland.jp/developer/download/image/SqueakPluginJ-2005-11-30.zip 
からダウンロード出来ます。特にまだ Linux や *BSD 版のインストーラがあ
りませんので、腕に覚えのある方は挑戦してみては如何でしょうか？

イメージは 
http://www.squeakland.org/installers/SqueakPluginInstaller.exe に含ま
れる SqueakPlugin.image に以下の操作を行った物です。

- 日本語フォントをインストール
- Utilities setUpdateServer: 'Squeakalpha' でアップデート元を変更
- #514 までアップデート
- ReleaseBuilderSqueakland new setupLanguageSpecifics を実行

p.s. Windows インストーラの為に山本さんの壁紙
http://www.thoruman.com/squeak/wallp/wp1.htm を使わせて頂きました。
